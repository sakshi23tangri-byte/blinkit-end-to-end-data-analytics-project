# Blinkit Customer & Product Analytics

## Project Overview

This project analyzes Blinkit’s customer behavior, product performance, and delivery efficiency using SQL, Power BI, Python, and Machine Learning.

## Tools Used

* SQL
* Python
* Power BI
* Pandas
* Matplotlib
* Scikit-learn

## Key Objectives

* Analyze customer purchasing behavior
* Track product sales and profitability
* Evaluate delivery performance
* Predict delivery delays using Logistic Regression
* Segment customers using K-Means Clustering

## Dashboard Pages

1. Home Overview
2. Customer Analytics
3. Product Analytics
4. Delivery & Feedback Analytics

## Machine Learning Models

### Logistic Regression

Used to predict whether an order will be delayed.

**Accuracy:** 90.9%

### K-Means Clustering

Segmented customers into:

* High-value customers
* Medium-value customers
* Low-value customers

## Key Insights

* 69.4% deliveries were on time
* High-value customers contribute significant revenue
* Product profitability varies across categories
* Customer ratings are mostly positive
* Revenue remains stable across months

## Business Recommendations

* Improve delayed delivery operations
* Reward high-value customers
* Upsell medium-value customers
* Optimize low-profit product categories
